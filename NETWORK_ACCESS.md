## 🚀 서버 실행 방법

### 1. 개발 서버 시작
```bash
cd client
npm run dev
```

서버가 실행되면 다음과 같은 메시지가 표시됩니다:
```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://10.201.10.151:5173/
```

### 2. 같은 네트워크의 다른 기기에서 접속
- **모바일 브라우저**를 열고 `http://10.201.10.151:5173` 입력
- **태블릿 브라우저**를 열고 `http://10.201.10.151:5173` 입력
- **다른 PC 브라우저**를 열고 `http://10.201.10.151:5173` 입력

## 📌 주의사항

1. **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다
2. **방화벽 설정**을 확인하세요 (포트 5173 허용 필요)
3. IP 주소는 **네트워크 환경에 따라 변경될 수 있습니다**
   - IP가 바뀌었다면 PowerShell에서 `ipconfig` 명령어로 확인

---

## 🔧 방화벽 문제 해결

만약 다른 기기에서 접속이 안 된다면:

### Windows 방화벽 허용 설정
1. **Windows 보안** > **방화벽 및 네트워크 보호** 열기
2. **고급 설정** 클릭
3. **인바운드 규칙** > **새 규칙**
4. **포트** 선택 > **TCP 5173** 입력
5. **연결 허용** 선택 후 완료

또는 간단하게:
```powershell
# PowerShell 관리자 권한으로 실행
New-NetFirewallRule -DisplayName "Vite Dev Server" -Direction Inbound -Protocol TCP -LocalPort 5173 -Action Allow
```

## 💡 IP 주소 확인 방법

현재 PC의 IP 주소가 변경되었을 경우:

**PowerShell:**
```powershell
ipconfig | Select-String "IPv4"
```

**CMD:**
```cmd
ipconfig | findstr IPv4
```
